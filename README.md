# Byers Lights - Installation Guide

A Stranger Things-inspired message wall with glowing bulbs.

## Quick Start

1. **Clone the repository:**

```bash
git clone https://github.com/satya-supercluster/upside-down-lights.git
cd byers-lights
```

2. **Install dependencies:**

```bash
npm install
```

3. **Run the development server:**

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Usage

- Click anywhere to start
- Type or click letters to create your message
- Click "Copy your message link" to share your message with others
